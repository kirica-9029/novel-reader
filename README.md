# Yomikaze

Web小説サイトを横断して、作品URL、更新確認、読了位置、お気に入り、メモを管理する静的PWAです。

## できること

- 作品URLの登録、編集、削除
- 原典サイトを新しいタブまたは現在タブで開く
- お気に入り、状態、タグ、読了位置、メモの保存
- RSS / Atom URLによる更新確認
- ブラウザ通知
- 設定画面
- JSONバックアップの書き出し、読み込み
- PWAとしてインストール可能

## 運用方針

このアプリは本文の全文再配信やミラーサイト化を目的にしていません。登録した原典URLを開く補助ツールとして動作し、RSS監視もユーザーが登録した公開フィードに対してのみ行います。

## 起動方法

`index.html` をブラウザで開くか、任意の静的サーバーで配信してください。

```powershell
python -m http.server 8080
```

その後、`http://localhost:8080` にアクセスします。
